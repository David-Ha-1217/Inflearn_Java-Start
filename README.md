# Inflearn_Java-start


## 프로젝트 소개
해당 프로젝트는 인프런에서 제공하는 **'김영한의 자바 입문'** 강의를 수강하고, 실습한 내용을 담고 있습니다. <br>
모든 실습 코드는 **Intellij IDEA**를 사용하여 작성되었습니다.


## 프로젝트 기간
* 2024년 6월 20일, 목요일 ~ 


## 프로젝트 목표
* 자바 프로그래밍의 기본 개념 및 문법 이해.
* Intellij IDEA를 활용한 자바 개발 환경 구축.
* 단계별 실습 과제를 통한 코딩 실력 향상.
* 깃허브를 활용한 코드 관리 및 공유.

## 실행 방법
1. 해당 Repository를 클론합니다.
2. IntelliJ IDEA를 열고 프로젝트를 로드합니다.
3. 코드를 빌드하고 실행합니다.


## 환경 설정
* Windows11 23H2
* IntelliJ IDEA 2024.1
* Java 17
